##### 单工，半双工，全双工

```
单工：数据传输只能一方发送，另一方接收
半双工：双方都可以发送和接收，但是在同一时间，只能有一方发送或接收
全双工：数据通信允许数据同时在两个方向上传输，同一时间可以同时接收和发送
```
